🏗️ Core Components Architecture

🔧 Wirehair Fountain Codes
O(N) erasure coding library that splits data into systematic and repair packets. Any N + 2% packets can recover the original file.
Key Features:
• Systematic packet generation
• Repair packet creation via XOR
• CRC32 integrity verification
• Configurable redundancy factor (default: 1.5x)
🔬 DCT Steganography
Discrete Cosine Transform embeds data in video frequency coefficients. Sign-bit manipulation survives YouTube's compression.
Mathematical Basis:
• Forward DCT: F = T · B · T^T
• Inverse DCT: B = T^T · F · T
• Sign-bit embedding (positive=0, negative=1)
• Threshold-controlled robustness
🎬 FFmpeg Engine
Professional video encoding/decoding pipeline. Handles RGB24 raw frames, H.264/FFV1 codecs, and streaming processing.
Capabilities:
• Raw video pipe streaming
• Lossless FFV1 codec support
• H.264 encoding for compatibility
• Frame extraction via FFprobe
🌐 YouTube Integration
OAuth 2.0 authenticated upload/download via YouTube Data API v3. Manages private videos and metadata.
Features:
• Resumable uploads
• Privacy control (unlisted/private)
• Metadata management
• Automatic retry logic
